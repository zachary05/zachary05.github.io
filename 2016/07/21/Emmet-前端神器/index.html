Emmet - 前端神器

Posted on 2016-07-21   |     |   visitors

Emmet 前端神器

官网给出的解释：

Emmet — the essential toolkit for web-developers

(Emmet - 超级重要的前端者的开发工具

Emmet is a plugin for many popular text editors which greatly improves HTML & CSS workflow

(Emmet 是一个插件，支持流行的文本编辑器，它能极大地提高 HTML & CSS 的编程流程。

总而言之，就是很牛逼。可以减少大量重复输入的操作。

使用方法也很简单：

  • 输入Emmet命令，按tab。
  • Emmet命令不能有空格，不能换行。

Emmet 常用方法总结

Emmet - HTML

  • ! // 自动生成HTML文档初始结构

    <!DOCTYPE html>
<html lang="en">
<head>
      <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>

</body>
</html>

  • link:css

    <link rel="stylesheet" href="style.css">

  • meta:vp 

    <meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">

  • div.i#i //自动生成带有类名和id名的标签 

    <div class="i" id="i"></div>

  • div.i#i+div.i1#i1 //‘+’代表兄弟标签

    <div class="i" id="i"></div>
<div class="i1" id="i1"></div>

  • div.i#i>div.i1#i1 //'>'代表子标签

    <div class="i" id="i">
     <div class="i1" id="i1"></div>
</div>

  • div.i#i>div.i1#i1^div.i2#i2 //'^'代表上级标签

    <div class="i" id="i">
      <div class="i1" id="i1"></div>
</div>
   <div class="i2" id="i2"></div>

  • a*3 //'*'代表重复次数

    <a href=""></a>
<a href=""></a>
<a href=""></a>

  • (div>a*3)+div //'()'代表分组，因此前面一组与后面的div是兄弟标签

    <div>
    <a href=""></a>
    <a href=""></a>
       <a href=""></a>
  </div>
  <div></div>

  • a[href="#" title="hi"] //'[attr]代表标签内的属性' 

    <a href="#" title="hi"></a>

  • li.i$*3 //'$'代表编号，自动生成编号

    <li class="i1"></li>
 <li class="i2"></li>
<li class="i3"></li>

  • li.i$@-*3 //'@-'代表倒序编号

    <li class="i3"></li>
  <li class="i2"></li>
  <li class="i1"></li>

  • li.i$@3*3 //'@n'代表从n开始编号

    <li class="i3"></li>
  <li class="i4"></li>
  <li class="i5"></li>

  • p{hello,world} //"{}"代表文本内容

    <p>hello,world</p>

  • input & button

    inp = input:t = input:text
input:p = input:password
input:c = input:checkbox
input:f = input:file
input:s = input:submit
input:i = input:image
input:b = input:button
btn:s = button:s = button:submit
btn:r = button:r = button:radio
btn:d = button:d = button:disabled

Emmet - CSS

  • w100

    width: 100px;

  • w100p

    width: 100%;

  • w100+w100p

    width: 100px;
  width: 100%;

  • m10-10 //用'-'连接多个属性

    margin: 10px 10px;

  • m10--10

    margin: 10px -10px;

  • m10em10em //如果带有单位则可以省略'-'

    margin: 10em 10em;

  • @

    @font-face {
      font-family:;
      src:url();
}

  • @f+ 

    @font-face {
     font-family: 'FontName';
      src: url('FileName.eot');
      src: url('FileName.eot?#iefix') format('embedded-opentype'),
     url('FileName.woff') format('woff'),
     url('FileName.ttf') format('truetype'),
     url('FileName.svg#FontName') format('svg');
      font-style: normal;
      font-weight: normal;
}

  • -border-radius //加上'-'可以自动生成前缀

    -webkit-border-radius: ;
  -moz-border-radius: ;
-ms-border-radius: ;
  -o-border-radius: ;
  border-radius: ;

  • lg(left,#83c9d2,50%,#4e7b85) //自动生成背景渐变

    background-image: -webkit-linear-gradient(left, #83c9d2, 50%, #4e7b85);
 background-image: -o-linear-gradient(left, #83c9d2, 50%, #4e7b85);
 background-image: linear-gradient(to right, #83c9d2, 50%, #4e7b85);

Emmet - Bonus

  • lorem //生成Lorem，并且可以指定数量，如lorem5

    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Recusandae voluptatem aperiam obcaecati consequatur veniam maiores temporibus, similique. Obcaecati a voluptate, fuga quia modi nostrum sint tenetur laboriosam voluptas porro eaque.

  • 隐式标签 //Emmet会自动根据父标签判定当前标签类别。

    默认为 div
ul 或 ol 下默认为 li
table 下默认为 tr
tr 下默认为 td
select 或 optgroup 下默认为 option

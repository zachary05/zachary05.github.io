Ajax学习笔记

Posted on 2016-07-18   |     |   visitors

AJAX是一种无须重新加载整个网页的情况下，能够更新部分网页的技术。

听起来挺酷，因为我知道，网页只有刷新才会呈现新的内容，但是Ajax的好处就是不用去刷新网页，就可以更新网站的局部内容。

XMLHttpRequest对象（XMLHttpRequest是Ajax的基础）

所有现代浏览器都支持XMLHttpRequest对象（IE5 IE6 使用ActiveXObject)。

XMLHttpRequest的作用：用于后台 [浏览器中的脚本(客户端)] 与服务器交换数据。这个对象充当着浏览器中中间人的角色。以往的请求都是由浏览器发出，而Javascript通过这个对象可以自己发送请求，同时也自己处理响应。

创建XMlHttpRequest对象

所有现代浏览器均内置XMLHttpRequest对象

var request = new XMLHttpRequest();

但，IE，你懂得，所以咱们得这样获取：（ActiveXObject是针对IE的）

if(window.XMLHttpRequest){
    var request = new XMLHttpRequest();
}else{
    var request = new ActiveXOject('Microsoft.XMLHTTP);
}

或者可以写一个函数当做脚本好了：

function getHTTPObject() {
  if (typeof XMLHttpRequest == "undefined") {
        XMLHttpRequest = function () {
          try {return new ActiveXObject("Msxml2.XMLHTTP.6.0");}
          catch (e) { }
          try { return new ActiveXObject("Msxml2.XMLHTTP.3.0"); }
          catch (e) { }
          try { return new ActiveXObject("Msxml2.XMLHTTP"); }
          catch (e) { }
        }
      }
  return new XMLHttpRequest();
}

Ajax 向服务器发送请求

主要使用XMLHttpRequest对象的open()和sent()方法。

request.open(method, url, async);

解释：

method：请求的类型：GET/POST

url：文件在服务器上的位置，该文件可以是任何类型，.txt/.xml/.asp/.php

async：true（异步）或者false（同步）。XMLHttpRequest对象如果要使用AJAX，open()方法的async参数必须设为true. 

//当请求为异步时，onreadystatechange事件中的就绪状态执行的函数

request.onreadystatechange = function(){
if(request.readyState == 4 && request.status == 200){
        doing something...
    }
}

request.send(string);

解释：
将请求发送到服务器

string：仅用于POST请求。

Ajax - onreadystatechange事件

请求发送到服务器之后，那么就会有基于响应的任务。并且有两个属性让我们知道请求怎么样了。

readyState和 status

每次readyState改变时，都会调用onreadystatechange事件。

readyState属性有5个值：

  • 0：请求未初始化
  • 1：服务器已经建立连接
  • 2：请求已接收
  • 3：请求已处理
  • 4：请求已完成

status属性的值有：

  • 200：’OK’表明成功
  • 301：Moved Permanently, 永久移动
  • 302：Move temporarily, 暂时移动
  • 304：Not Modified, 未修改
  • 307：Temporary Redirect, 暂时重定向
  • 401：Unauthorised, 未授权
  • 403：Forbidden, 禁止访问
  • 404：Not Found, 未找到页面
  • 500：Internal Server Error, 服务器发生错误

基本上之后2XX,和304的状态码表示服务器返回正常。

所以一般看

request.readyState == 4 && request.status == 200 才表明请求成功了

statusText属性是只读值，返回一个字符串，表明服务器发送的状态提示，如“200 OK”.

Ajax响应

竟然你请求了数据，那么服务器就会有响应回来的数据。

获取服务器的响应，要使用XMLHttpRequest对象的responseText属性或者responseXML属性。

responseText : 获取字符串形式的响应数据

responseXML : 获取XML形式的响应数据

响应成功后，我们就可以用服务器返回来的responseText/responseXML数据进行处理，显示给用户看。
